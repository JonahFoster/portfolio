Unit Converter

American Friendly!

While completing the Front-End Web Developer course through Scrimba, I was assigned this project with the following requirements:

  • Build it from scratch
  • Generate two random passwords when the user clicks the button.
  • Each password should be 15 characters long

This project was especially notable to me since it was the first time I built a JavaScript project more complex than simple counting. I also went above and beyond the project parameters, adding the ability to toggle on symbols and numbers in the generated passwords.
