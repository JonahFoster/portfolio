Endorsement App

You are a champion!

While completing the Front-End Web Developer course through Scrimba, I was assigned this project with the following requirements:

  • Build it from scratch
  • Follow the design
  • Use a Firebase Realtime Database
  • Update in realtime

I went beyond those parameters for my project, adding a like feature, reversing the order so newest endorsements appear first, and adding a to and from input. If I ever revisit this project in the future, I intend to remove the ability to set blank endorsements and limit likes to 1 per person.
