Basketball Scoreboard

Ball not included.

This was the second solo project I completed through the front-end developer course on Scrimba. I was assigned this project with the following requirements:

  • Build it from scratch
  • Folow the provided design
  • All six buttons should work

Unlike some of the other self-guided projects from Scrimba, I opted to not implement anything beyond the basic requirements. I was eager to continue learning, so I completed the project and moved on. If I revisit this project in the future, I will likely add a "New Game" button, a foul counter, quarters, and a box score.
